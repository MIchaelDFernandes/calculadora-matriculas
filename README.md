# Calculadora de Série para Matrícula Escolar

Esta é uma aplicação web simples para ajudar pais e responsáveis a determinar a série escolar correta para a matrícula de uma criança no ano letivo de 2026, com base em sua data de nascimento.

## Como Usar

1.  Abra o arquivo `index.html` em seu navegador.
2.  Clique no botão "Escolha uma data" para abrir o seletor de data.
3.  Selecione a data de nascimento da criança.
4.  A aplicação exibirá instantaneamente a série escolar e a etapa de matrícula correspondentes para o ano letivo de 2026.

## Funcionalidades

- **Cálculo Preciso da Série:** Utiliza as datas oficiais de matrícula de 2026 para fornecer a série correta.
- **Interface Amigável:** Uma interface limpa e simples para facilitar a seleção da data.
- **Resultados Instantâneos:** A série é exibida imediatamente após a escolha da data.
- **Tecnologia Moderna:** Construído com HTML, CSS e JavaScript modernos e sem frameworks.

## Estrutura do Projeto

- `index.html`: O arquivo HTML principal da aplicação.
- `style.css`: Contém todos os estilos para um layout limpo e responsivo.
- `main.js`: Gerencia a interação do usuário e a lógica de cálculo da série.
- `nascidos-2026.js`: Um módulo JavaScript que exporta os dados de matrícula para o ano letivo de 2026.
